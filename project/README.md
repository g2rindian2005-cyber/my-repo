# 📚 Student Management System

A simple and stylish student management frontend application.

## Features
- ✨ Modern, responsive UI with gradient design
- ➕ Add new students easily
- 📝 Edit student information
- 🗑️ Delete students
- 📊 View all students in a beautiful card layout
- 🔔 Real-time notifications

## How to Use

1. **Open the application** - Open `index.html` in your web browser
2. **Add a Student** - Fill in the form with:
   - Student Name
   - Email
   - Roll Number
   - Grade (A, B, or C)
   - Click "Add Student"
3. **Edit a Student** - Click the "Edit" button on any student card to update their details
4. **Delete a Student** - Click the "Delete" button to remove a student

## Color Scheme
- Primary: Purple gradient (`#667eea` to `#764ba2`)
- Edit Button: Green
- Delete Button: Red
- Background: Light white cards with shadows

## Sample Data
The app comes with 5 sample students pre-loaded for demonstration.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- Vanilla JavaScript (No dependencies!)

## Files
- `index.html` - Main structure
- `styles.css` - Styling and layout
- `script.js` - Functionality and data management
